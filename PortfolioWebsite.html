BandBuddy Short Internship Replicating Website

This project is a personal portfolio website, created to showcase my design and development skills. I started by forking an existing open-source template and completely reshaped it into my own minimal, editorial-style portfolio. The goal was to practice Tailwind CSS, improve code structure, and present my work in a professional way.

Development Progress

Initial Design

I used a template from GitHub as a base (GitHub project by Shivaraj-Kolekar), but I wanted to make the website truly my own. The initial idea was to create a portfolio that felt structured, clean and visually balanced. I was especially drawn to layouts with clear lines and a block-based structure. Something that feels both sharp and artistic. I fell in love with the style so i wanted to create my own.

Development

During development, I reorganized the code structure into clear, separate folders for components, assets, and pages. I also rewrote the styling using Tailwind CSS, which I learned specifically for this project. Thanks to my knowledge of HTML and CSS, I picked it up quickly and was able to build responsive layouts efficiently.

Results

The final website is clean and structured. Its block-based layout gives it an editorial and artistic feel that matches my personal style. The design focuses on strong visual lines and clear typography. This project not only showcases my technical skills but also reflects my eye for design and ability to turn a template into something personal and professional.

Reflections

This project helped me improve both my frontend development and design thinking. I learned to work more efficiently with Tailwind, organize a project properly and develop a consistent visual style. In future updates I plan to make the website responsive and add more interactive elements to make the site even more engaging.
